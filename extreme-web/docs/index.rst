Extreme UI documentation
========================================

Extreme UI is the open-source front-end for `Cryptonomex Extreme <https://github.com/cryptonomex/extreme>`_. A live hosted wallet running a custom themed version of Extreme UI for Extreme can be found at `<https://extreme.openledger.info>`_
